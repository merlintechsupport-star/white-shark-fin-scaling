**Stanford Hopkins Marine Station**

## What This Project Does
This project analyzes whether pectoral fin area scales proportionally with body size in eastern Pacific white sharks, and compares the results to Australian white shark data from the literature.

The core question: as a white shark grows bigger, do its pectoral fins keep up (isometric scaling), grow relatively larger (positive allometry), or fall behind (negative allometry)?

> **Note:** The current dataset uses simulated data. Real measurements will be pulled from Labelbox once the API key is available.

## How to Run the Analysis Pipeline

Run these scripts in order from the project root:

```bash
python analysis/step1_load.py       # 1. Load and validate data
python analysis/step2_plots.py      # 2. Generate raw plots
python analysis/step3_analysis.py   # 3. Run regressions
python analysis/step4_fitted_plots.py  # 4. Generate fitted plots
```

To launch the dashboard locally:

```bash
streamlit run app.py
```

## Data Format (`data/merged_shark_data.csv`)

| Column | Description |
|---|---|
| `TL_cm` | Total length of the shark (cm) |
| `left_pec_cm2` | Left pectoral fin area (cm²) |
| `right_pec_cm2` | Right pectoral fin area (cm²) |
| `avg_pec_cm2` | Average of left and right fin areas (cm²) |
| `sex` | Sex of the shark (M or F) |

## Regression Methods

- **OLS (Ordinary Least Squares):** Standard log-log regression. Gives the allometric exponent `b`.
- **SMA (Standardized Major Axis):** Preferred in biology for scaling studies. Accounts for error in both variables. Calculated as `b_SMA = b_OLS / r`.
- **Nonlinear Power Regression:** Fits the power law `area = a × length^b` directly.

A slope of `b = 2` means isometric scaling. Below 2 means fins grow relatively smaller as the shark gets bigger (negative allometry). The Australian literature value is `b = 1.68` (Kolborg et al. 2013).
